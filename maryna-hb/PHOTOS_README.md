# Інструкція по додаванню фотографій

## Як додати свої фотографії на сайт:

1. **Підготуйте 5 фотографій** у форматі JPG або PNG
   - Будь-який розмір - фото автоматично масштабуються без обрізання
   - Назвіть їх: `photo1.jpg`, `photo2.jpg`, `photo3.jpg`, `photo4.jpg`, `photo5.jpg`

2. **Помістіть фотографії** у папку `photos/`
   ```
   maryna-hb/
   └── photos/
       ├── photo1.jpg  <- Ваша перша фотографія (ліва верхня)
       ├── photo2.jpg  <- Ваша друга фотографія (права верхня)
       ├── photo3.jpg  <- Ваша третя фотографія (ліва нижня)
       ├── photo4.jpg  <- Ваша четверта фотографія (права нижня)
       └── photo5.jpg  <- Ваша п'ята фотографія (центральна)
   ```

3. **Готово!** Фотографії автоматично з'являться на столі в 3D сцені

## Якщо фотографії не відображаються:

- Перевірте, що файли називаються точно `photo1.jpg`, `photo2.jpg`, `photo3.jpg`
- Переконайтеся, що файли знаходяться в папці `photos/`
- Відкрийте консоль браузера (F12) і подивіться чи є помилки завантаження
- Якщо використовуєте локальний файл, запустіть локальний сервер:
  ```bash
  python -m http.server 8000
  ```

## Поточний стан:

✅ Папка `photos/` створена
✅ Код оновлено для завантаження фотографій
✅ Якщо фото не знайдено - буде показано золотий плейсхолдер

## Особливості:

✅ Фрейми тепер більші (0.9x0.9 метра) і краще видно на столі
✅ Фото квадратні та зберігають оригінальні пропорції
✅ Горизонтальні та вертикальні фото автоматично підганяються без обрізання
✅ 5 рамок розташовані по всьому столу для кращого огляду
